IMPLEMENTATION FILES:
====================
- /lib/utils/roleRedirect.ts - Contains the role-to-path mapping logic
- /lib/utils/protectedRoutes.ts - Contains route protection logic
- /lib/hooks/useAuth.ts - Updated login hook with role-based redirection

HOW IT WORKS:
=============
1. User logs in with their credentials
2. System extracts user role from JWT token
3. User is redirected to their role-specific page
4. Route protection prevents access to other roles' pages
5. Unauthorized access redirects users back to their own page
